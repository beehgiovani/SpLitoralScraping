# MetroMarGeo - Lista de Bairros (Metrópole & Mar)

Esta lista consolida os bairros oficiais e principais núcleos urbanos das cidades cobertas pelo ecossistema **MetroMarGeo**. Estas informações são essenciais para a filtragem e organização dos dados no CRM Imobiliário Interativo.

---

## **1. Guarujá (39 Bairros Oficiais)**
A cidade é dividida em 4 regiões principais, totalizando 39 bairros.

| Região | Bairros |
| :--- | :--- |
| **Centro** | Barra Funda, Centro, Pitangueiras, Vila Maia, Vila Baiana, Enseada. |
| **Vicente de Carvalho** | Jardim Boa Esperança, Jardim Casqueiro, Jardim Cunhambebe, Pae Cará, Vila Alice, Vila Áurea. |
| **Sul / Praias** | Guaiúba, Jardim Astúrias, Jardim Las Palmas, Santa Cruz dos Navegantes, Tombo. |
| **Norte / Expansão** | Balneário Cidade Atlântica, Jardim Mar e Céu, Jardim Virgínia, Perequê, Santa Rosa, Santo Antônio. |
| **Outros** | Morrinhos, Vila Edna, Vila Zilda, Cachoeira, Prainha Branca. |

---

## **2. Santos (47 Bairros/Núcleos)**
Santos possui uma divisão clara entre a Zona Leste (Orla), Noroeste e os Morros.

| Zona | Bairros |
| :--- | :--- |
| **Orla (Leste)** | Aparecida, Boqueirão, Embaré, Gonzaga, José Menino, Pompeia, Ponta da Praia. |
| **Intermediária** | Campo Grande, Encruzilhada, Estuário, Macuco, Marapé, Vila Belmiro, Vila Mathias. |
| **Centro / Portuária** | Centro, Paquetá, Valongo, Vila Nova, Outerinhos. |
| **Noroeste** | Areia Branca, Bom Retiro, Castelo, Chico de Paula, Rádio Clube, Santa Maria, São Manoel. |
| **Morros** | Morro da Nova Cintra, Morro do Jabaquara, Morro do José Menino, Morro Santa Maria. |
| **Área Continental** | Caruara, Monte Cabrão, Iriri. |

| Orla (Praias) | Interior / Rodovia |

| Canto do Forte, Boqueirão, Guilhermina, Aviação, Tupi, Ocian, Mirim, Maracanã, Caiçara, Real, Flórida, Solemar. | Anhanguera, Antártica, Glória, Melvi, Nova Mirim, Quietude, Ribeirópolis, Samambaia, Tupiry, Vila Sônia. |

## **4. Peruíbe (Principais Bairros)**
Peruíbe possui uma vasta área com muitos loteamentos e balneários.

*   **Centro / Orla:** Centro, Arpoador, Balneário Stella Maris, Costão, Guaraú.
*   **Balneários:** Balneário Flórida, Balneário São João Batista, Jardim Caraguatá, Jardim Ribamar.
*   **Interior:** Estância dos Nobres, Recreio Santista, Vila Erminda.

---

## **5. Bertioga (Incluindo Riviera)**
Bertioga é marcada por grandes condomínios e áreas de preservação.

*   **Principais:** Centro, Albatroz, Maitinga, Rio da Praia, São Lourenço, Indaiá, Vista Linda.
*   **Destaque:** **Riviera de São Lourenço** (Módulo 1 ao 33).
*   **Norte:** Boraceia, Guaratuba.

---

## **6. São Paulo - GeoSampa (Distritos Relevantes)**
Foco nos distritos centrais e de maior interesse imobiliário.

*   **Centro:** Sé, República, Consolação, Bela Vista, Santa Cecília.
*   **Zona Sul:** Vila Mariana, Moema, Itaim Bibi, Santo Amaro, Morumbi.
*   **Zona Oeste:** Pinheiros, Perdizes, Lapa, Butantã.
*   **Zona Leste:** Tatuapé, Mooca, Belém, Anália Franco.

---

**Nota:** Esta lista é baseada nos cadastros municipais e pode sofrer alterações conforme novas leis de zoneamento. Para o **GuaruGeo**, utilize estas chaves para indexar os imóveis extraídos.
